# jukebox-randomizer
A multi-class Java project that simulates a simple jukebox system. It demonstrates object-oriented design with classes such as Record, JukeInterface, and JukeTester.

## Features
- Stores a collection of Record objects (title and artist)
- Displays all records in the jukebox
- Selects and prints random records
- Checks if a specific record exists in the jukebox
- Finds the slot/index of a record
- Removes records from the jukebox
- Demonstrates object-oriented design with multiple classes

## How to Run
- Runs on:
- OnlineGDB
- Replit
- Any local Java IDE

## Compile
javac JukeInterface.java JukeTester.java Record.java Jukebox.java

## Run
java JukeTester
